About Project:

This is a dashboard that displays informative charts about bacteria that live inside the human's belly button from sample of hundreds of people. As can be seen, the dashboard allows to visualize each person's demographic information, top 10 bacteria cultures found, Bacteria cultures per sample, belly button washing frequency. User can select ID no. and the data is displayed.
